# 🍳 Forkify - Recipe Search Application

A modern, fully-featured recipe search and bookmarking application built with vanilla JavaScript. Search through over 1,000,000 recipes, bookmark your favorites, and even upload your own recipes!

## Image

## ✨ Features

- 🔍 **Search Recipes** - Search through 1,000,000+ recipes by ingredients or dish name
- 📖 **Detailed Recipe View** - View cooking time, servings, ingredients list, and directions
- 📑 **Pagination** - Navigate through search results with intuitive pagination
- ⭐ **Bookmarks** - Save your favorite recipes locally (persists across sessions)
- ➕ **Add Custom Recipes** - Upload your own recipes with ingredients and instructions
- 📱 **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- 🎯 **Servings Adjuster** - Automatically scale ingredient quantities when adjusting servings
- 👤 **User-Generated Recipes** - Special visual indicator for user-uploaded recipes

## 🛠️ Built With

- **JavaScript (ES6+)** - Core application logic with modern JavaScript features
- **Parcel** - Module bundler and build tool
- **SCSS** - Styled with modular, maintainable SCSS
- **Forkify API** - Recipe data API service
- **Core.js** - Polyfills for legacy browser support
- **Regenerator Runtime** - Support for async/await syntax

## 📦 Installation & Setup

**Clone the repository**

```bash
git clone https://github.com/huzaifaghazali/forkify.git
cd forkify
npm install
npm start
```
